# XVideoView
封装VideoView，支持横竖屏，支持音量和亮度调节

<br>

###背景：
>最近开发SDK，原APP项目中为支持RTMP封装了B站的IJKPlyaer，但是太大；UniversalVideoView合适，但还不支持音量和亮度调节，于是重新封装了一套。
